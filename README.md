# 🧪 Backend Clean Architecture – Prueba Técnica

Este repositorio contiene el desarrollo de una API REST utilizando **Node.js**, **Express** y **TypeScript**, como parte de una **prueba técnica**. El proyecto sigue principios de **Clean Architecture** y está preparado para escalar con autenticación JWT, documentación Swagger, Redis para caching y conexión a bases de datos como MySQL.

---

## 🛠️ Tecnologías principales

- **Node.js** + **Express**
- **TypeScript**
- **Arquitectura limpia (Clean Architecture)**
- **Swagger** (documentación)
- **JWT** (autenticación)
- **Redis** (caching)
- **MySQL** (base de datos relacional)
- **Dotenv** (manejo de variables de entorno)
- **Jest** (pruebas unitarias e integración – próximamente)

---

## 🚀 Como levantar el servidor

Se recomienda usar node v.18.20.3.
**Consideraciones:** levantar un contenedor de mysql y redis 
Despues de clonar el repositorio ejecutar los siguientes comandos
- **1.** npm install.
- **2.** npm run dev
